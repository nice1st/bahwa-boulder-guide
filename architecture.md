# 바위 볼더링 가이드 — 아키텍처 문서

## 뷰잉 (일반 사용자)

### 지도
- 마커 종류별 아이콘 구분 (볼더/주차장/화장실/갈림길)
- 볼더 마커: 썸네일 포함, GPS 좌표 표기
- 줌 인/아웃에 따라 마커 클러스터링 (겹침 처리)
- **필터**: V 난이도 (복수 선택, V10+ 묶음), 최소 패드 수
- 어프로치 경로(Path) 폴리라인 — 기본 숨김
- 내 위치 버튼 (GPS)
- 뒤로가기(popstate) — 루트 상세 → 마커 패널 → 지도 순서로 단계적 닫기
- 상단바 — 앱 이름, 햄버거 메뉴 (로그인/로그아웃, admin만 관리자 링크)

### 마커 클릭
- 바텀시트 — 루트 목록 + 사진 슬라이드
- "어프로치" 버튼 → 경로 폴리라인 토글

### 루트 클릭
- 영상 링크 전체 목록
- 댓글 목록 + 댓글 작성 (비로그인)

---

## 관리자 기능

- `/admin` 경로, Google OAuth + role=admin 체크
- PC 친화 레이아웃: 좌(지도 50%) + 우(목록+편집 50%)
- 사용자 지도와 동일한 KakaoMap 컴포넌트 공유 (mode='admin')

### 마커 관리
- 사진 업로드 → EXIF GPS 자동 추출 → 50m 반경 감지
- 지도 클릭으로 좌표 지정
- 마커 종류 선택 (볼더/주차장/화장실/갈림길)

### 루트 관리
- 볼더 마커에 루트 추가 (이름, V난이도, 설명, 사진, 영상 링크)

### 경로(Path) 관리
- 지도에서 마커/좌표 순서대로 클릭 → 폴리라인 생성

### 사용자 관리
- 사용자 목록 조회, 역할 승격/강등 (user/admin)
