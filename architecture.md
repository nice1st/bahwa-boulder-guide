# 바위 볼더링 가이드 — 아키텍처 문서

## 프로젝트 개요

지도 기반 볼더링 루트 가이드 웹앱.
관리자가 스팟/루트를 등록하고, 일반 사용자는 조회 및 댓글을 남긴다.

---

## 핵심 기능

| 기능 | 대상 | 인증 |
|------|------|------|
| 지도에서 스팟 마커 조회 | 모든 사용자 | 불필요 |
| 스팟 상세 + 루트 목록 조회 | 모든 사용자 | 불필요 |
| 루트 상세 (사진, 설명, 영상 링크) | 모든 사용자 | 불필요 |
| 댓글 작성/조회 | 모든 사용자 | 불필요 |
| 스팟/루트 등록·수정·삭제 | 관리자 | 필요 |

---

## 데이터 모델

### Marker (지도 마커)
```
id, type (boulder|parking|toilet|junction), name, lat, lng,
thumbnail_url, created_at
```
- type별 아이콘 구분 표시

### Route (볼더 루트) — type=boulder 마커에 속함
```
id, marker_id, name, grade (V0~V17), description,
min_pads (최소 패드 수), photo_urls[], video_urls[], created_at
```

### Path (어프로치 경로 라인)
```
id, name, waypoints (좌표 배열 JSON), created_at
```
- waypoints: 마커 ID 목록 또는 자유 좌표 배열
- 마커 간 연결(e2e) 또는 직접 그린 폴리라인 모두 지원

### Comment (댓글)
```
id, route_id, content, created_at
```

## 뷰잉 (일반 사용자)

### 지도
- 마커 종류별 아이콘 구분 (볼더/주차장/화장실/갈림길)
- 볼더 마커: 썸네일 포함, GPS 좌표 표기
- 줌 인/아웃에 따라 마커 클러스터링 (겹침 처리)
- **필터**: V 난이도, 최소 패드 수
- 어프로치 경로(Path) 폴리라인 — 기본 숨김

### 마커 클릭
- 루트 목록 + 사진 넘겨보기(슬라이드) 패널 표시
- 루트 목록 항목: 대표 사진, 이름, 난이도, 최소 패드 수, 첫 번째 영상 링크
- "어프로치" 버튼 → 해당 마커의 경로 라인 연하게 표시 / 다시 누르면 숨김 (마커당 경로 1개)

### 루트 클릭
- 영상 링크 전체 목록
- 댓글 목록 + 댓글 작성 (비로그인)

## 관리자 기능

- `/admin` 경로, Supabase Auth로 보호

### 마커 관리
- 사진 업로드 → EXIF GPS 자동 추출 → 50m 반경 내 기존 마커 검색
  - 근처 마커 있음: 기존 마커에 추가 or 새 마커 생성 선택
  - 근처 마커 없음 or GPS 없음: 지도에서 직접 핀 찍기
- 마커 선택 후 사진 추가 (역방향)
- 마커 종류 선택 (볼더/주차장/화장실/갈림길)

### 루트 관리
- 볼더 마커에 루트 추가 (이름, V난이도, 설명, 사진, 영상 링크)
- 사진 Supabase Storage 업로드

### 경로(Path) 관리
- 지도에서 마커 순서대로 클릭 → 폴리라인 생성
- 중간 경유 좌표 추가 가능 (마커 없는 꺾임점)
